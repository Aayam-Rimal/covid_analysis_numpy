# COVID-19 Analysis

This program analyzes covid 19 data on different metrics obtained from the csv file. It uses numpy as the primary dependency. Calculates basic statistical functions over different fields.

## Features

- Pulls data from CSV instead of raw arrays defined within the program
- Analyzes mean,median,standard deviation of different metrics like deaths,confirmed,recovered and active cases
- Calculates minimum and maximum values
- Calculates correlation between different variables

## Installation and running

1. Clone the repository:
- Git clone git@github.com:Aayam-Rimal/covid_analysis_numpy.git

2. Create a virtual environment(recommended)
- python 3 -m venv venv

3. Activate the virtual environment
- source venv/bin/activate 

4. Install required dependencies
- pip install -r requirements.txt

5. Run in your local machine 
- python covid_analysis.py
